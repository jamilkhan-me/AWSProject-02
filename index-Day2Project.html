30 Days of AWS · Day 2 of 30

Cloud Infrastructure Deployed.

This page is served globally from Amazon S3 via CloudFront CDN — no servers, no OS to patch, 400+ edge locations, and a 99.99% SLA. Built by Jamil Khan as part of the #30DaysOfAWS challenge.

aws s3 sync ./site s3://jamil-day2-website --delete
Origin

Amazon S3

Static assets stored in an S3 bucket with versioning enabled. Bucket is private — accessible only via CloudFront OAC.

CDN

CloudFront

Global CDN with 400+ edge locations. HTTP auto-redirects to HTTPS. Free TLS cert via ACM. Sub-10ms cache hits.

Security

Origin Access Control

OAC policy ensures only this CloudFront distribution can read from S3. Bucket is never exposed to public internet.

DNS

Route 53

Alias record pointing to the CloudFront distribution. Zero-latency DNS resolution with health check failover.

bucket-policy.json · CloudFront OAC 
{
  "Version": "2012-10-17",
  "Statement": [{
    "Sid": "AllowCloudFrontServicePrincipal",
    "Effect": "Allow",
    "Principal": { "Service": "cloudfront.amazonaws.com" },
    "Action": "s3:GetObject",
    "Resource": "arn:aws:s3:::jamil-day2-website/*",
    // Only THIS CloudFront distribution can access the bucket
    "Condition": {
      "StringEquals": {
        "AWS:SourceArn": "arn:aws:cloudfront::ACCOUNT:distribution/ID"
      }
    }
  }]
}
